# HCD Linode Deployment

This repository provides Terraform scripts for deploying **HCD (Highly Concurrent Database) on Linode** using:

- **Linode Kubernetes Engine (LKE)**
- **Linode Object Storage**
- **Mission Control for Management**
- **DataStax API Deployment**

## Features
- Terraform configuration for deploying Linode Kubernetes clusters
- Automated Object Storage setup
- Cluster-wide observability using Loki, Prometheus, and Mimir
- Pre-configured API deployment for Cassandra workloads
- Step-by-step deployment guides
